# Library-API

Library api is a api for CRUD your books.

## Installation

1.install xampp for mysql database [https://www.apachefriends.org/] 

2.install this framework on terminal for using this project

go get -u github.com/gin-gonic/gin 
go get -u gorm.io/gorm 
go get -u gorm.io/driver/mysql


## Usage

// First you start Apache and MySQL on XAMPP Application
// Second you make a new database in localhost/phpmyadmin then create a new database with name library
// After all step you can go to terminal on your project and type go run main.go
// Finish:)
